## 신규 기능

### 전역 콤보 XP 시스템 (Global Combo XP)
- 연속 정답 콤보가 퀴즈 세션을 넘어 **전역적으로 유지**됩니다.
  - 앱을 재시작하거나 새 세션을 시작해도 이전 콤보가 이어집니다.
  - 오답 시 콤보 즉시 초기화 (광고로 힌트 사용 유도 강화)
- 연속 정답 XP 보너스 **최대 +5XP로 상향**:
  | 연속 정답 | 기존 보너스 | 변경 후 보너스 |
  |-----------|------------|----------------|
  | 2연속     | +1 XP      | +1 XP          |
  | 3연속     | +1 XP      | +2 XP          |
  | 4연속     | +1 XP      | +3 XP          |
  | 5연속     | +1 XP      | +4 XP          |
  | 6연속 이상 | +1 XP     | +5 XP (최대)   |
- 퀴즈 화면 콤보 팝업에서 실시간으로 `+N XP` 보너스 표시
- 홈 화면 상단 바에 콤보 2 이상일 때 `N연 / 콤보` 뱃지 표시

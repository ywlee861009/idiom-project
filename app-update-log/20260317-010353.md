# 🚀 Idiom Quiz Update Log (v1.0.7-11)

## 📅 날짜: 2026-03-17

## 🛠️ 주요 변경 사항

### 1. 🧠 출제 알고리즘 고도화 (Smart Selection)
- **미수집 성어 우선 출제**: 이미 정답을 맞힌(수집된) 성어보다 아직 맞히지 못한 성어를 우선적으로 출제하여 '서고(수집)'를 채우는 재미를 강화했습니다.
- **무작위성 강화**: 출제 후보군(Shuffle Pool)을 기존 대비 10배 확대하여 문제 노출의 반복 패턴을 제거했습니다.

### 2. 💾 데이터 동기화 로직 개선 (Progress Preservation)
- **스마트 병합(Smart Merge)**: 새로운 성어 데이터를 가져올 때 기존 진행 상황(노출 횟수, 정답 횟수)을 초기화하지 않고 메타데이터만 업데이트하도록 수정했습니다. 이제 앱 업데이트 후에도 수집한 성어 기록이 안전하게 유지됩니다.

### 3. 📈 사용자 통계 및 UX 개선
- **학습수 vs 수집수 괴리 해결**: 전체 학습 횟수 대비 수집량이 낮았던 문제를 알고리즘 개선을 통해 해결했습니다.
- **연속 정답 로직 최적화**: 퀴즈 세션 내 정답 판정 및 경험치 획득 로직을 안정화했습니다.

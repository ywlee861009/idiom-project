# 사자성어 퀴즈 프로젝트 요구사항 (REQUIREMENTS)

## 1. 프로젝트 목표 (Project Goal)
사용자가 사자성어 퀴즈를 풀고 정답을 맞춤으로써 점수를 획득하는 재미있는 학습용 애플리케이션 개발.

## 2. 핵심 기능 (Core Features - MVP)
- [ ] **사자성어 데이터**: 최소 20개 이상의 사자성어와 그 의미를 담은 데이터셋 구축.
- [ ] **퀴즈 엔진**: 랜덤하게 사자성어를 제시하고 정답 여부를 판단하는 로직.
- [ ] **점수 시스템**: 정답 시 점수 가산, 오답 시 점수 차감 또는 유지.
- [ ] **결과 화면**: 현재 점수 및 최종 획득 점수 표시.

## 3. 선택 기능 (Optional Features - Nice-to-have)
- [ ] 난이도 선택 (초급, 중급, 고급)
- [ ] 오답 노트 기능 (틀린 사자성어 다시 보기)
- [ ] 랭킹 시스템 (로컬 파일 또는 DB 저장)
- [ ] 멀티플레이어 모드

## 4. 기술 스택 (Technical Stack - Draft)
- **Language**: Python (CLI 기반 앱 우선 제안)
- **Data Format**: JSON (사자성어 데이터 관리)
- **Platform**: Terminal / Console
